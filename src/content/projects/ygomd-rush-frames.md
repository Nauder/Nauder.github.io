## Overview

A runtime mod for Yu-Gi-Oh! Master Duel that transforms the game's card visuals to match the Rush Duel aesthetic. Using MelonLoader and the Harmony patcher, it hooks into the game at startup and automatically repositions card elements.

## Features

- Converts Master Duel card frames to Rush Duel style
- Repositions artwork, level indicators, ATK/DEF values, attribute icons, and text areas
- Replaces traditional level stars with numeric level display
- Applies automatically on launch with no extra configuration

## Stack

- **C# / .NET 6** — mod language and runtime
- **MelonLoader** — mod loader framework for Unity games
- **Harmony** — runtime method patching
